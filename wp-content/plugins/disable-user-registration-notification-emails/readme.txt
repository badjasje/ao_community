Turns off the notification sent to the admin email when a new user account is registered. Works with WP >= 4.6.0.

== Description ==

When this plugin is activated, it disables the notification sent to the admin email when a new user account is registered.

If you like this plugin, please consider [making a donation](https://potentplugins.com/donate/?utm_source=disable-user-registration-notification-emails&utm_medium=link&utm_campaign=wp-plugin-repo-donate-link).

== Installation ==

1. Click "Plugins" > "Add New" in the WordPress admin menu.
2. Search for "Disable User Registration Notification Emails".
3. Click "Install Now".
4. Click "Activate Plugin".

Alternatively, you can manually upload the plugin to your wp-content/plugins directory.
